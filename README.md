## Installation

gems and packages (gems require ruby 2.1.2)
```shell
$ bundle install && bower install && npm install
```

## Development

**working files are within the ``app`` dir**

local server task
```shell
$ grunt serve
```

build task
```shell
$ grunt build
```

deployment task
```shell
$ grunt deploy
```
